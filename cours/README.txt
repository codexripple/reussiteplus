Placez ici les fichiers de cours pour chaque matière dans un sous-dossier nommé selon la matière (ex: Mathématiques, Français, etc.).

Exemple de structure :

cours/
  Mathématiques/
    fractions.pdf
    video_chapitre1.mp4
    exercices.docx
  Français/
    conjugaison.pdf
    audio_lecon1.mp3
    liens.txt

Tous les formats sont acceptés : PDF, vidéo, audio, images, PowerPoint, Word, liens, etc.

Les fichiers ajoutés via l'admin seront automatiquement placés dans le bon dossier.
